Upload default media for this club.

You can choose to upload one default track to be played in all executions by all troops representing this club, or one track per discipline.

The track selection is hierarchical, which means that in a tournament, we will first look for a track uploaded for that specific tournament in that specific discipline. That is configured when you assign your troops to a competition. If none is found, we will look for a track for a specific discipline, which you configure in this panel. If we still cannot find a track, we will look for a club track.
